# INVERSE VERSION 1: DARK NEON (Inverse of Light Professional)

## Color Scheme
- **Background:** #0A0A0A (Near Black)
- **Text:** #FFFFFF (Pure White)
- **Accent:** #00FF88 (Electric Green/Neon)
- **Secondary:** #1A1A2E (Deep Navy)
- **Glow:** #00FF88 at 30% opacity

## Vibe
Cyberpunk meets local business. High energy, futuristic, stands out from boring corporate sites.

## Design Elements
- Dark mode everything
- Neon green buttons with glow effects
- Sharp, angular cards
- Monospace fonts for headlines
- Subtle grid patterns in background
- High contrast for accessibility

## Recommended For
Business owners who want to look cutting-edge. Great for younger entrepreneurs, tech-forward services, or anyone wanting to break from "boring local business" stereotypes.

## CSS Variables
```css
--bg-primary: #0A0A0A;
--bg-secondary: #1A1A2E;
--text-primary: #FFFFFF;
--text-secondary: #B0B0B0;
--accent: #00FF88;
--accent-glow: rgba(0, 255, 136, 0.3);
--border: #2A2A3E;
```

## Typography
- **Headlines:** Space Grotesk or JetBrains Mono (bold, uppercase)
- **Body:** Inter or System UI
- **Accents:** Monospace for buttons/CTAs

## Button Styles
- Background: transparent
- Border: 2px solid #00FF88
- Text: #00FF88
- Hover: Background fills with #00FF88, text turns black
- Box-shadow: 0 0 20px rgba(0, 255, 136, 0.4) on hover

## Card Styles
- Background: #1A1A2E
- Border: 1px solid #2A2A3E
- Border-radius: 4px (sharp, not rounded)
- Hover: Border color changes to #00FF88
