# INVERSE VERSION 2: LIGHT GOLD (Inverse of Dark Bold)

## Color Scheme
- **Background:** #FFFFFF (Pure White)
- **Text:** #1A1A1A (Soft Black)
- **Accent:** #D4AF37 (Classic Gold)
- **Secondary:** #F8F6F1 (Warm White/Cream)
- **Highlight:** #FFF8DC (Cornsilk)

## Vibe
Premium luxury meets approachability. Think high-end hotel lobby or premium financial services—professional but warm.

## Design Elements
- Clean white backgrounds
- Gold accents on buttons and icons
- Elegant serif fonts for headlines
- Subtle cream/tan secondary backgrounds
- Thin gold borders and dividers
- Lots of breathing room

## Recommended For
Professional services, consultants, real estate, financial advisors—anyone wanting to signal "premium" without being stuffy.

## CSS Variables
```css
--bg-primary: #FFFFFF;
--bg-secondary: #F8F6F1;
--bg-accent: #FFF8DC;
--text-primary: #1A1A1A;
--text-secondary: #5A5A5A;
--accent: #D4AF37;
--accent-light: #F4E4BC;
--border: #E8E4D9;
```

## Typography
- **Headlines:** Playfair Display or Cormorant Garamond (elegant serif)
- **Body:** Inter or Source Sans Pro (clean sans-serif)
- **Accents:** Uppercase with letter-spacing for labels

## Button Styles
- Background: #D4AF37
- Text: #1A1A1A (dark for contrast)
- Border: none
- Hover: Background darkens to #B8960C
- Subtle shadow: 0 4px 14px rgba(212, 175, 55, 0.3)

## Card Styles
- Background: #FFFFFF
- Border: 1px solid #E8E4D9
- Border-radius: 8px
- Box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04)
- Hover: Border color changes to #D4AF37
